# zennta概要

## zenntaはどんなサービス？

qiita・zennを参考に作成したブログサービスです。

## zenntaのページ

- 会員登録/ログイン
- 記事一覧ページ
- ランキングページ
- 管理ページ
  - 記事作成/編集ページ
- マイページ
  - 編集ページ
  - ブックマーページ

## zenntaの機能

- 記事作成/編集/削除/非表示
- カテゴリ/タグ
- お気に入り
- 良いね

## 他やりたいこと

- Docker開発環境構築
- PHPUnitでのテストコード
- AWSを利用してデプロイ
- CircleCIの導入
- Selenium+duskでのE2Eテスト
